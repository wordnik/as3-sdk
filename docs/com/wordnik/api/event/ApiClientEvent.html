Packagecom.wordnik.api.event
Classpublic class ApiClientEvent
InheritanceApiClientEvent flash.events.Event

Event dispatched by the SDK to communicate success events and failure events. If a custom dispatcher has been assigned by the consumer on the WordnikClient then the dispatcher dispatches the ApiClientEvent to indicate success or failure of the invocation using the Response
